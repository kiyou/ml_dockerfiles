# ML Dockerfiles

Dockerfiles for pytorch machine learning environments with some packages

## Base Image

- [pytorch/pytorch](https://hub.docker.com/r/pytorch/pytorch)

## Python Packages

- matplotlib
- h5py
- numpy
- optuna
- pandas
- pickle5
- pymatgen
- sklearn
- tqdm
- psycopg2
